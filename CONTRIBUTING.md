# Guia de Contribuição

Agradecemos seu interesse em contribuir com o Projeto Raia! Toda contribuição é bem-vinda e fundamental para o sucesso desta iniciativa.

## Como Começar

1.  **Faça um Fork** do repositório para a sua conta do GitHub.
2.  **Clone o seu fork** para a sua máquina local
3.  **Crie uma nova branch** para suas alterações:
    ```bash
    git checkout -b minha-contribuicao
    ```

## Fazendo Alterações

1.  Faça as modificações desejadas no código.
2.  Adicione seus arquivos e faça o commit das alterações com uma mensagem clara:
    ```bash
    git add .
    git commit -m "feat: Adiciona nova funcionalidade de percepção"
    # Use "fix:" para correções, "docs:" para documentação, etc.
    ```
3.  Envie suas alterações para o seu fork no GitHub:
    ```bash
    git push origin minha-contribuicao
    ```

## Enviando um Pull Request

1.  Abra a página do seu fork no GitHub.
2.  Clique no botão "New pull request".
3.  Certifique-se de que a branch base seja a `main` do repositório original e a `head` seja a sua branch `minha-contribuicao`.
4.  Adicione um título e uma descrição clara para o seu Pull Request, explicando o que foi alterado e por quê.
5.  Clique em "Create pull request".

Após o envio, um dos mantenedores irá revisar seu código e, se tudo estiver correto, suas alterações serão integradas ao projeto principal.
